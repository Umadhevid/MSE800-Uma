Student Management Project:

Description:

This project manages the student and lecturer details of Yoobee College.
It provides options to add, delete, and view student and lecturer records.

Project Structure:

student_manager.py – Contains create, view, delete, and search student functionality

lecturer_manager.py – Contains create, view, delete, and search lecturer functionality

main.py – The main program to run the application

database.py – Contains the database connection and table creation logic

Prerequisites:

Python 3.8 or above

SQLite3 (included with Python)

Usage:

Run main.py

Choose an option from the menu to:

Add a student or lecturer

View student or lecturer records

Delete a student or lecturer

Operations:

Add – Insert a new record

View – Display all records

Delete – Remove a record by ID
